Creating Articles

Articles work a bit different from our previous sites. Our new articles are interactive, visual ways to teach concepts.

Prerequisites:

First, ensure that you have an IDE that allows you to write in the CSS and HTML languages. Visual Studio Code by Microsoft is recommended. You can download it from the following link below:

(Do note that this project does not require much coding, if any at all, it's all just creating a layout for each article page.)

Next, download our template guide, which will allow us to drag and drop your article into our website, so that it looks similar to all the other pages on our site. To open this file, first, you need to unzip the file. Either right click on the downloaded folder and click extract/unzip file if you're using Windows, or double click on your download to unzip it on a Mac. Then, open Visual Studio Code, and drag the file inside the folder you unzipped into Visual Studio Code. It should open a tab with the template.

Layout Basics for Your Article:

Each part of the article page can be customized based on what content you want to insert into the website. To see examples of types of content you can add to the page, download our Article Feature Example, and open it with Visual Studio Code in the same way you did for the template. This will show you how to use each feature in the website.

Copy and paste one of the features in the V brackets, and add your own text to it by following the instructions in the green comments. Then put them into the article template (the first file linked at the top of the page), by pasting each feature into the spot with the green comment.

Special Math Features:

For those writing math and physics articles, you have access to a few extra features.

Firstly, you will have access to the MathJax library, which will allow you to make math equations appear correctly. You will also have access to Desmos, allowing you to include Desmos in your articles. Please contact Joel if you would like to use Desmos, as he will need to give you access to the API key to unlock the software.

You can find sample code for both these features here. You may copy and paste this into the code, but make sure to use our API key if you are using Desmos, and make sure to not copy the comments from the code (these will likely appear green).
