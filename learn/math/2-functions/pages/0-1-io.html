What is a function?

In simple terms, a function is a way for us to take an input, modify it, and produce an output, using the same method every time. Think back to patterns you may have learned when you were younger.

For example, say we have a pattern in which for each term, the number increases by 5 (we'll assume the term number and the number start at 0).

If we are looking for term 1, we produce `5`, and term 2 produces the number 10, which is the same as `5+5`, which is also the same as `2(5)`. The table below represents the pattern:

Term NumberProduced NumberProduced Number
`1``5``5(1)`
`2``10``5(2)`
`3``15``5(3)`
`4``20``5(4)`
`5``25``5(5)`
`6``30``5(6)`
`7``35``5(7)`
`8``40``5(8)`

If we continue this pattern, we notice something. If we multiply each term number by 5, we get the number in the pattern for that term. This is an example of a function, modifying an input to get an output.

We can represent our new function as `y=5x`. The x and y variables are most commonly used (you'll find out why later), but as you can see, the x variable is our input in this function, and the y variable is the output in this function.

If we plug our term number into `x` and use our new function, we will get the number in our pattern (`y`).

Important: If you've never seen expressions like 5(8) before, note that in Math, we can represent multiplication as 2 numbers next to each other in brackets instead of using a multiplication sign. This is because the variable letters (like `x`) become common.

So `5(8)` is equal to `5*8`, which is equal to 40.

When variables like `x` and `y` are put next to each other, or next to another number, they are multiplied

For example, `5x = 5*x` and `xy=x*y`

What do we use functions for?

Functions can be used for many different purposes. You'll use functions across all major scientific fields, and also everywhere in Math, like probability and statistics.

On the next page, we'll learn more about how we can put our functions on a graph!

Try and write a function for this pattern. For every term (increase in `x`), add 14.

To get into the habit of using variables, let's call our input, or term number `x`, and our output, or produced number `y`. If we add 14 each time, then our first term will be `14`, and our second term will be `28`, which is the same as `14(2)`.

As you can see, when we multiply our input by 14, we get our output. Therefore, we can write the function as `y=14x` (you can use any letters that you like for the variables).
