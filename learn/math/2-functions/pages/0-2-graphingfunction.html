Graphing Functions

Now that we know where functions come from, we need a way to visualy represent them, so we can more easily find information. One thing we can do is graph our function, which will show us the relationship between both variables.

If you remember from the previous page, we named our variables `x` and `y`. You'll also notice that on our coordinate plane, we have an x-axis, and a y-axis. Usually, we graph the input along the x-axis, and the output along the y-axis.

Coordinates have an x-component, which tells us how far the point is moved from 0 in the horizontal direction, and a y-component, which tells us how far the point is moved from 0 in the vertical direction.

If we make an input the x-component, and the output a y-component, you'll see that we start to produce a line.

Let's look at the following graph below. As you'll notice this function is `y=ax`. The `a` can be adjusted by using the slider, to represent any number between -10 and 10 that `x` can be multiplied by to retrieve `y`. Try and move the a slider to get the line to pass through the point `(1,6)` on the graph:

Now, replace a in our function `y=ax` with the number you set your slider to on the graph above to pass through the point.

Try plugging in the value 1 for `x` from the coordinate on the graph into your function. You'll notice that you will get 6 as `y` (if your a value is correct), which matches the point on your coordinate plane! Isn't that neat?

This works with every coordinate along the line, meaning that by looking at the graph, you can tell what the other points will be, and what output each input would produce!

This means that for every function, as long as there is 1 unknown variable on each side, you can try different inputs to produce different outputs.

Now let's look at a second graph below. But now, there's a second variable (b)! Try and find 4 different values of `a` and `b` that will allow the line to pass through the point on the graph `(1, 6)`:

Again, if you create a function `y=ax+b`, and substitute those values of a and b that allowed the line to pass through the point, if you substitute 1 for `x` (from the point on the graph), you'll get 6 for `y`!

However, with each of your different a and b values, if you put them onto your graph, you'll notice that the lines don't pass through the same points. Some lines will have a lower slope (think of this like adding less to each input in your pattern). This results in a lower output, which is why the slope is smaller.

Now that we've shifted our thinking away from simple patterns and towards functions, try some examples before you move on to learning about different types of functions!

If we have the function `y=2x+3`, and `x=2`, what is `y`?

Since we know `x` is equal to 2, we can substitute `x` for 2 in our function. Therefore `2(2)+3=7`, and `y=7`.

If the points `(0,0)`, `(1,2)`, and `(2,4)` are on the graph, what function would create a line that would pass through all 3? (You can use one of the interactive graphs above to help you).

We can see the x-coordinates of our points as inputs (0, 1, and 2), and the y-coordinates as outputs (0, 2, 4). If you notice, each input can be multiplied by 2 to get each output.

Therefore, the function that will pass through all 3 points is `y=2x`.

If the points `(0,0)`, `(1,3)`, `(2,5)`, and `(3,7)` are on the graph, what function would create a line that would pass through all 4? (You can use one of the interactive graphs above to help you).

Again, we can see the x-coordinates of our points as inputs (0, 1, 2, and 3), and the y-coordinates as outputs (0, 3, 5, and 7). If you notice, each input can be multiplied by 2 and added to 1 to get each output.

Therefore, the function that will pass through all 4 points is `y=2x+1`.
